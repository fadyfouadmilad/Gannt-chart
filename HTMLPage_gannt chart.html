CPM Rules & Formulas

Forward Pass Calculation
ES = Max(EF of all Predecessors)
EF = ES + Duration
Start with initial activities (no predecessors): ES = 0. For subsequent activities, ES is the maximum EF of all its predecessors.
Backward Pass Calculation
LF = Min(LS of all Successors)
LS = LF - Duration
Start from project end: LF = Max(EF). For final activities, LF = Project Completion Time. Work backwards to calculate LS and LF.
Float Calculations
TF = LF - EF = LS - ES
FF = Min(ES of Successors) - EF
Total Float (TF) is the maximum delay without affecting project completion. Free Float (FF) is the delay without affecting successor activities.
Critical Path Rules
Critical Path = Activities where TF = 0
Activities on the critical path have zero total float. Any delay in these activities will delay the entire project completion.
Network Rules (A.O.N)
Activity-on-Node (A.O.N) Method
Each activity is represented by a node. Arrows show dependencies. No looping or circular dependencies allowed in the network.
Gantt Chart Rules
Bar spans from ES to EF
Float shown as extension
Activities are shown as horizontal bars. Critical path activities are highlighted. Float is shown as lighter extension beyond EF.
